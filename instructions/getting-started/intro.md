Welcome to the **Scripts and Files Demo** lab!

In this lab you will learn how Instruqt lifecycle scripts work by completing two hands-on tasks:

1. **Create a file** — use the terminal to create a file, then let Instruqt verify it exists.
2. **Configure an app** — edit a configuration file that was provisioned from the `files/` directory.

Use the terminal on the right to run commands. When you're ready, move to the next page.
