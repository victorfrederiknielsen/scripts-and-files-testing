## Create a File

Your first task is simple: create a file called `hello.txt` in the `/tmp` directory.

You can do this with:

```bash
echo "Hello, Instruqt!" > /tmp/hello.txt
```

Or simply:

```bash
touch /tmp/hello.txt
```

Once the file exists, click **Check** to validate your work.
